Help

How to download module?
  1. Click Course in Dashboard
  2. Choose module
  3. Click module’s name to download
  4. System will download the file and you will see the module in your local device.
How to upload task?
  1. Click Task Menu
  2. Choose task
  3. Read the details of task
  4. Click “Choose File” Button, then choose your file from your local device
  5. Click “Submit” Button to submit your file
